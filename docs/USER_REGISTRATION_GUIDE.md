## â“ Common Issues & Solutions

### "OTP not received"
**Possible causes:**
- Network delay (wait 1-2 minutes)
- Wrong phone number entered
- SMS blocked by carrier

**Solutions:**
1. Check your spam/blocked messages
2. Try "Resend OTP" button (appears after 60 seconds)
3. Verify your number has the +91 country code
4. Check your mobile signal strength

### "Invalid phone number"
**Requirements:**
- Must start with country code (+91 for India)
- Must be exactly 10 digits after country code
- No spaces or special characters

**Correct format:** `+919876543210`
**Wrong formats:** 
- âŒ `9876543210` (missing +91)
- âŒ `+91 98765 43210` (has spaces)
- âŒ `+91-9876543210` (has dash)

### "PERMISSION_DENIED: Missing or insufficient permissions"
This error has been fixed in the latest version. If you see this:
1. Make sure you're using the latest APK
2. Clear app data: Settings â†’ Apps â†’ ROSTRY â†’ Storage â†’ Clear Data
3. Uninstall and reinstall the app
4. If still persists, contact support

### "Sending..." stuck / App frozen
**Solutions:**
1. Check your internet connection (WiFi or mobile data)
2. Try switching between WiFi and mobile data
3. Force close the app and restart
4. Clear app cache
5. If problem persists after 2 minutes, restart your phone

### "App won't install"
**Solutions:**
1. Make sure you have enough storage (at least 100MB free)
2. Enable "Install from unknown sources" in Settings
3. Try uninstalling any old version first
4. Download the APK again (it might be corrupted)
